# Marcus Cyber Toolkit Hub

An open-source cybersecurity resources hub featuring tools, frameworks, and documentation for learning, research, network discovery, web application testing, security validation, and OSINT investigations.

## Features

- **Responsive Design**: Clean, modern interface with a cybersecurity theme
- **Search Functionality**: Filter tools in real-time
- **Tool Categories**: Organized by security discipline
- **GitHub Pages Ready**: Easily deployable on GitHub Pages

## File Structure

```
cyber-toolkit-hub/
├── index.html          # Main HTML file
├── styles/
│   └── style.css       # Stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── README.md           # This file
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/marcusnp11-cyber/cyber-toolkit-hub.git
   ```

2. Open `index.html` in your browser to view locally

## GitHub Pages Deployment

This repository is configured for GitHub Pages. To enable it:

1. Go to your repository settings
2. Navigate to **Pages** section
3. Select `main` branch as the source
4. Your site will be published at: `https://marcusnp11-cyber.github.io/cyber-toolkit-hub/`

## Tools Included

- **Burp Suite Community Edition** - Web application security testing
- **Nmap** - Network discovery and mapping
- **Kali Linux** - Security-focused operating system
- **Metasploit Framework** - Security validation framework
- **OSINT Framework** - Open-source intelligence gathering
- **GitHub Security Repositories** - Community security projects

## Customization

You can easily add more tools by:

1. Adding new card elements in `index.html`
2. Modifying colors in `styles/style.css`
3. Updating the search logic in `js/script.js`
